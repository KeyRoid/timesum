Privacy Policy

プライバシーポリシー

TimeSum は、ユーザーの個人情報を一切収集しません。

収集する情報

本アプリは以下を含むいかなる情報も収集・送信しません。

  • 個人情報
  • 利用状況データ
  • 位置情報
  • 端末識別情報

通信について

TimeSum は通信機能を使用しません。

データ保存

計算履歴はアプリ起動中のみ保持され、終了時に破棄されます。

お問い合わせ

お問い合わせは GitHub リポジトリ経由で行ってください。

Privacy Policy

TimeSum does not collect any personal information.

Information Collection

This app does not collect or transmit:

  • Personal data
  • Usage analytics
  • Location data
  • Device identifiers

Network Usage

TimeSum does not use network communication.

Data Storage

Calculation history is kept only during app runtime and discarded when the app is closed.
